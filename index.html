From Abstract Models to Concrete Code

Architecture used to be semi-structured abstractions about separate concerns.  These abstractions were visually communicated using various graphical conventions (UML Diagrams, ER Diagrams, Network Diagrams). 

Architecture has now evolved into orchestrated configurations of solution building blocks.   These orchestrated configurations enable agility and scalability of the build, integration and deployment of solutions using a network of DevOps pipelines.
